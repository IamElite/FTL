## 💻 Section 2: Bot Commands

Here is the list of commands you can use with the bot.

### 👤 User Commands
| Command | Description |
| :--- | :--- |
| `/start` | Initialize the bot and activate tokens. |
| `/link [n]` | Generate links for the replied file and next `n` files. |
| `/dc` | Get Data Center information for a user/file. |
| `/ping` | Check bot latency and status. |
| `/help` | Detailed usage instructions. |

### 🔑 Admin Commands
| Command | Description |
| :--- | :--- |
| `/status` | View system uptime and resource distribution. |
| `/broadcast` | Send announcements to all users. |
| `/stats` | Monitor usage analytics. |
| `/ban` / `/unban` | Manage user/channel access. |
| `/restart` | Safely restart the bot instance. |
| `/log` | Fetch latest execution logs. |
